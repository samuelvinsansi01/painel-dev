# V21 — Base Integrada

Esta versão une a base do CRM novo com a estrutura antiga de redirecionamentos/prospecção.

## Base principal
- `index.html`
- `css/styles.css`
- `js/app.js`
- `js/adapters/supabase-adapter.js`

## Trazido do redirect-main
- `/api`
- `/public/r.html`

Nenhuma regra de negócio do CRM foi alterada para evitar quebra.
